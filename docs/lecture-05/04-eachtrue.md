## 🔹 მარტივი მაგალითი — სტრიქონზე:

```ts
@MinLength(3)
name: string;
```

👉 ამ შემთხვევაში `name` უნდა იყოს მინიმუმ 3 სიმბოლო.
მაგალითად:

- ✅ `"Tom"` — სწორი
- ❌ `"Al"` — არასწორი (მოკლეა)

---

## 🔹 `{ each: true }` — როცა მასივია

თუ შენი ველი არის **მასივი**, მაგ.:

```ts
@MinLength(3, { each: true })
tags: string[];
```

ეს ნიშნავს:

- რომ **მასივი შეიძლება იყოს როგორც გინდა დიდი ან პატარა**
- მაგრამ თითოეული ელემენტი შიგნით უნდა იყოს მინიმუმ 3 სიმბოლო

მაგალითად 👇

✅ სწორი:

```ts
tags = ['dev', 'api', 'blog'];
```

❌ არასწორი:

```ts
tags = ['go', 'ai', 'js'];
// "go" და "js" მხოლოდ 2 სიმბოლოა → validation error
```

---

## 🧠 ანუ მოკლედ:

| პარამეტრი        | რას აკეთებს                                    |
| ---------------- | ---------------------------------------------- |
| `@MinLength(3)`  | სიგრძე უნდა იყოს ≥ 3 სიმბოლო                   |
| `{ each: true }` | თუ მასივია, ყველა ელემენტზე ცალ-ცალკე შეამოწმე |
